# Claude Code ガイドライン

このプロジェクトでClaude Codeが従うべきガイドラインです。

## コーディング規約

- コードは読みやすく、保守しやすいものにしてください
- 適切なコメントを追加してください
- 既存のコードスタイルに従ってください

## プロジェクト構造

- すべてのコードは適切なディレクトリに配置してください
- 新しい機能を追加する際は、既存の構造を尊重してください

## テスト

- 可能な限りテストを追加してください
- 既存のテストが失敗しないようにしてください

## コミットメッセージ

- 明確で説明的なコミットメッセージを使用してください
- 変更の目的を説明してください
